Тестирование шоколадного яйца
описание объекта: дано шоколадное яйцо "Киндер сюрприз" (стандартное) в фольгированной обертке,
на упаковке написано, что внутри яйца находится игрушка из новогодней коллекции.

тест кейс 1 : Яйцо фирмы "Киндер сюрприз"
шаги                            			ожидаемый результат
- посмотреть на обертке	наименование		- название соответствует
  и производителя		

тест кейс 2 : Яйцо обернуто в фольгу, а не другой материал
шаги                            			ожидаемый результат
- развернуть обертку						- материал - фольга
- оценить материал на качество
 (осмотреть, укусить)
- подвергуть термообработке
 (поджечь)

тест кейс 3 : Параметры яйца - стандартные
шаги                            			ожидаемый результат
- сделать замеры							- соответствует
- взвесить

тест кейс 4 : Начинка - шоколадная
шаги                            			ожидаемый результат
- посмотреть состав на обертке				- яйцо шоколадное
- развернуть обертку
- оценить яйцо визуально
- понюхать
- попробовать на вкус

тест кейс 5 : Качество шоколада
шаги										ожидаемый результат
- изучить состав шоколада					- настоящий шоколад
- срок хранения
- развернуть обертку
- далее оценить по :
  звуку (хрусту шоколада)
  внешнему виду (должен блестеть)
  запаху (пахнуть щоколадом)
- попробовать на вкус

тест кейс 6 : Наличие предупреждения о несъедобных объектах
шаги                           				ожидаемый результат
- изучить обертку 							- предупреждение написано 

тест кейс 7 : Наличие игрушки
шаги                            			ожидаемый результат
- потрясти яйцо								- игрушка присутствует
- взвесить
- вскрыть яйцо

тест кейс 8 : Игрушка из новогодней коллекции
шаги                            			ожидаемый результат
- развернуть обертку						-игрушка из новогодней коллекции
- вскрыть шоколад
- осмотреть игрушку визуально
- сравнить с каталогом коллекций
  (при наличии)

тест кейс 9 : Вытащить игрушку не повредив обертку
шаги                            			ожидаемый результат
- аккуратно развернуть обертку				- нет, визуально видны следы вскрытия
- разломить шоколадное яйцо на 2 половинки
- вытащить игрушку
- соединить 2 половинки шоколада
- завернуть обратно в фольгу

тест кейс 10 : При какой температуре растает шоколадное яйцо
шаги                            			ожидаемый результат
- оставить яйцо при комнатной температуре	-шоколад должен растаять при определенной t (например, 20 градусов)
(на солнце, на горячей плите,
 долго держать в руке)

тест кейс 11 : Надкусить яйцо в обертке
шаги                            			ожидаемый результат
- не разворачивая обертку, откусить			- во рту будут кусок фольги, шоколада и зубная боль от укуса игрушки
  
тест кейс 12 : Сварить в кастрюле
шаги                            			ожидаемый результат
- включить плиту							- шоколад расстает, останется вареная игрушка (не поврежденная)
- взять кастрюлю
- налить воды в кастрюлю
- поставить кастрюлю на плиту
- дождаться закипания воды
- поместить яйцо в воду на 5 мин.
 (вместе с оберткой)

тест кейс 13 : Уронить яйцо
шаги                            			ожидаемый результат
- бросить яйцо на пол с расстояния 1м        - яйцо должно разломиться, но игрушка уцелеть
(бросить яйцо с окна 2 этажа на асфальт и тд)

тест кейс 14 : Сдавыить яйцо
шаги										ожидаемый результат
- сдавить любыми способами					- шоколад растрескается

тест кейс 15 : Дать яйцо ребенку
шаги                            			ожидаемый результат
- дать яйцо ребенку             			- ребенок без труда должен развернуть обертку,
- наблюдать                                  разломить шоколад (съесть его) и получить игрушку
